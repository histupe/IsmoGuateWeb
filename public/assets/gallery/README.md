# Galería de Fotos de Ismo Guate

Guarda aquí las fotos reales de tus eventos y productos personalizados.
Para que se reflejen en la página web, asegúrate de que los nombres de los archivos coincidan con los configurados en `src/App.tsx` o actualiza el código allí.

Nombres sugeridos:
- `foto1.jpg`
- `foto2.jpg`
- `foto3.jpg`
- `foto4.jpg`
- `foto5.jpg`
- `foto6.jpg`

Puedes agregar más fotos y actualizar el array `galleryImages` en `src/App.tsx`.
